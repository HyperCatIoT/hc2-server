Catalogue Crawler

This is a crawler and visualisation tool for IoT catalogues in the application/vnd.tsbiot.catalogue.json format. Given a starting URL, it will crawl any linked catalogues then render a directed graph of discovered relationships. The nodes and edges shown in the graph can be filtered.

Select an example catalogue from the box below, or enter the URL of any public catalogue and click "crawl".
You may also browse the example catalogue files. For a description of what each catalogue shows, see examples/README.txt.
